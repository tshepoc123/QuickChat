
## Project Description
QuickChat is a Java console-based chat registration and login system.

The application allows a user to:
- Register an account
- Validate username rules
- Validate password complexity
- Validate South African cell phone number
- Login successfully after registration

---

## Features Implemented
✔ User Registration  
✔ Username Validation  
✔ Password Complexity Check  
✔ Cell Phone Number Validation  
✔ Login Authentication  
✔ Unit Testing using JUnit  

---

## How to Run the Project
1. Open project in Apache NetBeans
2. Right-click project
3. Select **Run**
4. Follow console prompts

---

## Testing
JUnit tests were implemented to verify:
- Valid username
- Invalid username
- Valid password
- Invalid password
- Valid cell phone number
- Login success and failure

Run tests:
Right-click project → **Test**
